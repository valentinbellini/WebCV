RS485 with DMA in FRDM-KL46Z

Using the Cortex M0+ microcontroller on the FRDM-KL46Z development board in order to establish RS-485 serial communication with another device. The system's operation is modeled using the Finite State Machine / UML Statecharts formalism. The message exchange between the two devices is implemented using circular queue data structures for reception and DMA-handled transactions for transmission. The application was programmed and debugged using the MCUXpresso environment and the associated libraries. Here is the link to GitHub
